¿Qué es MetricTime?

MetricTime es una aplicación web que permite la gestión de diferentes métricas asociadas a uno o varios dispositivos electrónicos, esto con el objetivo de controlar y monitorear el comportamiento a través de gráficos presentados en tiempo real, de forma que el usuario pueda realizar los diferentes análisis y observaciones de las diferentes métricas reportadas por los agentes de monitorización (cada uno de los dispositivos electrónicos).

Visión general de MetricTime

Este proyecto tiene como objetivo permitir a los usuarios finales realizar un monitoreo de las distintas métricas que están asociadas a los dispositivos electrónicos con los que este cuente de tal forma que pueda llevar a cabo un control y seguimiento de su comportamiento de una manera sencilla y efectiva, el producto final será una aplicación web donde se brindará al usuario final una forma intuitiva de manejar el sistema y poder conectarse de forma rápida para comenzar a realizar el respectivo control y seguimiento de sus datos.

¿Por qué elegirnos?

MetricTime es una aplicación web Open Source que está desarrollada para utilizar de manera gratuita. Parte del propósito de MetricTime es que en cualquier persona desde cualquier lugar pueda monitorear dispositivos IoT. Este proyecto tiene un alcance muy amplio y está muy relacionado con las tecnologías que se encuentran en auge; entonces MetricTime pretende ayudar al desarrollo de las ciudades inteligentes y la implementación de dichas tecnologías en los países que lo requieran.

Inicio de sesión o registro

MetricTime requiere la creación de una cuenta (nickname, email y contraseña) para poder hacer uso del software.

Página principal

Al iniciar sesión correctamente, el usuario es redireccionado a la página principal o dashboard en donde podrá navegar por el menú y en caso de tener agentes conectados se tendrá la vista previa del dashboard del agente.

Historial

En el menú de navegación el usuario tiene la opción del historial que como su nombre indica, muestra el historial de agentes conectados e incluso puede realizar la busqueda de algún agente en específico.

Instrucciones

En el menú de navegación el usuario tiene la opción de instrucciones, esta opción muestra el paso a paso de como agregar un nuevo agente a MetricTime, estas instrucciones se muestran en un pdf creado previamente.

Editar información personal

Al dar clic en la foto de perfil, el usuario puede ingresar a ver la información personal y modificarla.

Regenerar Token

Al dar clic en la foto de perfil, el usuario puede configurar un nuevo token o regeneralo en caso de haberlo creado anteriormente.

Datos Extra

MetricTime es un aplicativo web compatible con los navegadores basados en chrome, firefox, edge, opera y safari. De igual manera, a continuación se muestran algunas características de como fue desarrollado MetricTime.

  • Características técnicas

    MetricTime se desarrolló en Windows y Linux (Ubuntu), es compatible con los SO Windows, Unix y para móviles

  • Tecnologías utilizadas

    Para el desarrollo de MetricTime se utilizó Springboot, nodeJS, ReactJS, PostgreSQL, NextJS, TailwindCSS y mosca

  • Comunidad

    MetricTime agradece a toda la comunidad de usuarios que nos den su apoyo recomendándonos y dando una buena opinión de la herramienta.

  • Metas futuras

    MetricTime pretende ser el aplicativo web más famoso de Latinoamérica para la gestión de dispositivos electrónicos.
